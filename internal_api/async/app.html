ichnaea.async.app¶

Holds global celery application state and startup / shutdown handlers.

ichnaea.async.app.celery_app = Internal module global holding the celery app.[source]¶

This is also the Celery app public endpoint, used on the command line via:

bin/celery -A ichnaea.async.app:celery_app <worker, beat>
ichnaea.async.app.init_worker_process(signal, sender, **kw)[source]¶

Called automatically when celery worker is started. This is executed inside each forked worker process.

Calls ichnaea.async.config.init_worker().

ichnaea.async.app.shutdown_worker_process(signal, sender, **kw)[source]¶

Called automatically when celery worker is stopped. This is executed inside each forked worker process.

Calls ichnaea.async.config.shutdown_worker().
