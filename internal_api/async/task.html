apply(*args, **kw)[source]¶

This method is only used when calling tasks directly and blocking on them. It’s also used if always_eager is set, like in tests.

If always_eager is set, we feed the task arguments through the de/serialization process to make sure the arguments can indeed be serialized into JSON.
