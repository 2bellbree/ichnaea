ichnaea.geocalc¶

Contains helper functions for various geo related calculations.

ichnaea.geocalc.add_meters_to_latitude(lat, distance)[source]¶

Return a latitude in degrees which is shifted by distance in meters.

The new latitude is bounded by our globally defined ichnaea.constants.MIN_LAT and ichnaea.constants.MAX_LAT.

ichnaea.geocalc.add_meters_to_longitude(lat, lon, distance)[source]¶

Return a longitude in degrees which is shifted by distance in meters.

The new longitude is bounded by our globally defined ichnaea.constants.MIN_LON and ichnaea.constants.MAX_LON.

ichnaea.geocalc.bound(low, value, high)[source]¶

If value is between low and high, return value. If value is below low, return low. If value is above high, return high. If low is below high, raise an exception.

ichnaea.geocalc.centroid(points)[source]¶

Compute the centroid (average lat and lon) from a set of points (two-dimensional lat/lon array).

Uses ichnaea._geocalc.centroid() internally.

ichnaea.geocalc.circle_radius(lat, lon, max_lat, max_lon, min_lat, min_lon)[source]¶

Compute the maximum distance, in meters, from a (lat, lon) point to any of the extreme points of a bounding box.

ichnaea.geocalc.distance(lat1, lon1, lat2, lon2)[source]¶

Compute the distance between a pair of lat/longs in meters using the haversine calculation. The output distance is in meters.

Uses ichnaea._geocalc.distance() internally.

ichnaea.geocalc.estimate_accuracy(lat, lon, points, minimum)[source]¶

Return the maximum range between a position (lat/lon) and a list of secondary positions (points). But at least use the specified minimum value.

ichnaea.geocalc.location_is_in_country(lat, lon, country_code, margin=0)[source]¶

Return whether or not a given (lat, lon) pair is inside one of the country subunits associated with a given alpha2 country code.

ichnaea.geocalc.maximum_country_radius(country_code)[source]¶

Return the maximum radius of a circle encompassing the largest country subunit in meters, rounded to 1 km increments.
