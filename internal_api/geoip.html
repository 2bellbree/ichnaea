ichnaea.geoip¶

Helper functions and classes around GeoIP lookups, based on Maxmind’s maxminddb and geoip2 Python packages.

ichnaea.geoip.configure_geoip(filename, mode=0, raven_client=None, _client=None)[source]¶

Configure and return a GeoIPWrapper instance.

If no geoip database file of the correct type can be found, return a GeoIPNull dummy implementation instead.

Parameters:
  • raven_client (raven.base.Client) – A configured raven/sentry client.
  • _client – Test-only hook to provide a pre-configured client.
ichnaea.geoip.geoip_accuracy(code, city=False)[source]¶

Return the best accuracy guess for the given GeoIP record.

Parameters:
  • code (str) – A two-letter ISO country code.
  • city (bool) – Do we have a city record or a country record.
Returns:

An accuracy guess in meters.
Return type:

int
class ichnaea.geoip.GeoIPWrapper(filename, mode=0)[source]¶

A wrapper around the geoip2.database.Reader class with two lookup functions which return None instead of raising errors.

Takes the absolute path to a geoip database on the local filesystem and an additional mode, which defaults to maxminddb.const.MODE_AUTO.
