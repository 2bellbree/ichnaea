ichnaea.region¶

Contains helper functions for region related tasks.

class ichnaea.region.Geocoder(json_file='/opt/mozilla/ichnaea/ichnaea/regions.geojson')[source]¶

The Geocoder offers reverse geocoding lat/lon positions into region codes.

region(lat, lon)[source]¶

Return a alpha2 region code matching the provided position. If the position is not found inside any region return None.

any_region(lat, lon)[source]¶

Is the provided lat/lon position inside any of the regions?

Returns False if the position is outside of all known regions.

in_region(lat, lon, alpha2)[source]¶

Is the provided lat/lon position inside the region associated with the given alpha2 region code.

in_region_mcc(lat, lon, mcc)[source]¶

Is the provided lat/lon position inside one of the regions associated with the given mcc.

regions_for_mcc(mcc, names=False)[source]¶

Return a list of alpha2 region codes matching the passed in mobile country code.

If the names argument is set to True, returns a list of genc.regions.Region instances instead.

The return list is filtered by the set of recognized ISO 3166 alpha2 codes present in the GENC dataset and those that we have region shapefiles for.

region_for_cell(lat, lon, mcc)[source]¶

Return a alpha2 region code matching the provided mcc and position. If the position is not found inside any region return None.

ichnaea.region.region_max_radius(code)[source]¶

Return the maximum radius of a circle encompassing the largest region subunit in meters, rounded to 1 km increments.
