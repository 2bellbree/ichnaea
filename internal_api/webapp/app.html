ichnaea.webapp.app¶

Holds global web application state and the WSGI handler.

ichnaea.webapp.app.wsgi_app(environ, start_response)[source]¶

Actual WSGI application endpoint, used on the command line via:

bin/gunicorn -c python:ichnaea.webapp.settings             ichnaea.webapp.app:wsgi_app

At startup reads the app config and calls ichnaea.webapp.config.main() once to setup the web app stored in the ichnaea.webapp.app._APP global.

ichnaea.webapp.app._APP = None[source]¶

Internal module global holding the runtime web app.
