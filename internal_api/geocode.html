class ichnaea.geocode.Geocoder(json_file='/opt/mozilla/ichnaea/ichnaea/regions.geojson.gz')[source]¶

The Geocoder offers reverse geocoding lat/lon positions into region codes.

region(lat, lon)[source]¶

Return a region code matching the provided position. If the position is not found inside any region return None.

any_region(lat, lon)[source]¶

Is the provided lat/lon position inside any of the regions?

Returns False if the position is outside of all known regions.

in_region(lat, lon, code)[source]¶

Is the provided lat/lon position inside the region associated with the given region code.

in_region_mcc(lat, lon, mcc)[source]¶

Is the provided lat/lon position inside one of the regions associated with the given mcc.

regions_for_mcc(mcc, metadata=False)[source]¶

Return a list of region codes matching the passed in mobile country code.

If the metadata argument is set to True, returns a list of dictionaries containing additional metadata instead.

The return list is filtered by the set of recognized region codes present in the GENC dataset.

region_for_cell(lat, lon, mcc)[source]¶

Return a region code matching the provided mcc and position. If the position is not found inside any region return None.

region_max_radius(code)[source]¶

Return the maximum radius of a circle encompassing the largest region subunit in meters, rounded to 1 km increments.
