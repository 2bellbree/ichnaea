ichnaea.country.country_for_location(lat, lon)[source]¶

Return a ISO alpha2 country code matching the provided location. If the location is found inside multiple or no countries return None.

ichnaea.country.country_matches_location(lat, lon, country_code, margin=0.0)[source]¶

Return whether or not a given (lat, lon) pair is inside one of the country subunits associated with a given alpha2 country code.

ichnaea.country.country_max_radius(country_code)[source]¶

Return the maximum radius of a circle encompassing the largest country subunit in meters, rounded to 1 km increments.
