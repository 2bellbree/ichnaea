ichnaea.models.api¶

class ichnaea.models.api.ApiKey(**kwargs)[source]¶

ApiKey model.

The allow_fallback and fallback columns determine if and what fallback location provider should be used.

The url specifies the external endpoint supporting the Geolocate API.

Requests to the fallback service can optionally be rate limited. Two settings control the rate limit:

ratelimit specifies how many requests are allowed to be made.

ratelimit_interval specifies the interval in seconds for which the ratelimit number applies, so for example one could configure 60 requests per 60 seconds, or 86400 requests per 86400 seconds (one day). Both would on average allow one request per second.

Finally the fallback service might allow caching of results inside the projects own Redis cache. cache_expire specifies the number of seconds for which entries are allowed to be cached.
