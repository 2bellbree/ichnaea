ichnaea.models.content¶

ichnaea.models.content.DATAMAP_GRID_STRUCT = <Struct object>[source]¶

A compact representation of a lat/lon grid as a byte sequence.

Consists of two unsigned 32 bit integers to encode both the scaled latitude and longitude each moved to the positive space by adding 90000 / 180000.

classmethod shard_id(lat, lon)[source]¶

Given a lat/lon return the correct shard id for this grid.

The world is split into four shards which each have similar amounts of populated land mass in them. Splitting the world at the equator / prime meridian would result in extremely unbalanced shard sizes.
