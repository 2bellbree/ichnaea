Model and schema related common classes.

Common MySQL database settings.

class ichnaea.models.base.BaseModel[source]¶

A base model with a common set of database settings.

class ichnaea.models.base.HashableDict(**kw)[source]¶

A class representing a unique combination of fields, much like a namedtuple. Instances of this class can be used as dictionary keys.

class ichnaea.models.base.ValidationMixin[source]¶

A mixin to tie a class and its valid colander schema together.

classmethod validate(entry, _raise_invalid=False, **kw)[source]¶

Returns a validated subset of the passed in entry dictionary, based on the classes _valid_schema, otherwise returns None.

class ichnaea.models.base.CreationMixin[source]¶

A mixin with a custom create constructor that validates the keyword arguments before creating an instance of the class.

classmethod create(_raise_invalid=False, **kw)[source]¶

Returns an instance of this class, if the passed in keyword arguments passed schema validation, otherwise returns None.
