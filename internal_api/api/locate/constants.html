ichnaea.api.locate.constants¶

ichnaea.api.locate.constants.EARTH_CIRCUMFERENCE = 40000000.0[source]¶

Approximate circumference of the Earth in meters.

ichnaea.api.locate.constants.MAX_WIFI_CLUSTER_METERS = 500[source]¶

Maximum distance between WiFi networks to be considered close enough to be from one consistent observation.

ichnaea.api.locate.constants.MIN_WIFIS_IN_QUERY = 2[source]¶

Minimum number of WiFi networks in a query to allow returning results based on WiFi information.

ichnaea.api.locate.constants.MIN_WIFIS_IN_CLUSTER = 2[source]¶

Minimum number of WiFi networks in a close-by cluster to allow returning results based on the cluster.

ichnaea.api.locate.constants.MAX_WIFIS_IN_CLUSTER = 5[source]¶

Maximum number of WiFi networks used from one combined cluster to form the aggregate result.

class ichnaea.api.locate.constants.DataAccuracy[source]¶

Describes the possible and actual accuracy class of a locate query.

Instances of this class can be compared based on their value or can be compared to int/float values.

These values are related to CELL_MIN_ACCURACY and GEOIP_CITY_ACCURACY and adjustments in one need to be reflected in the other.

high = None[source]¶

High accuracy, probably WiFi based.

medium = None[source]¶

Medium accuracy, probably cell based.

low = None[source]¶

Low accuracy, large cell, cell area or GeoIP.
